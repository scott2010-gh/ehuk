# 에흑() <br> <img src="https://img.shields.io/badge/Esolang-686868?style=for-the-badge&logo=GitHub&logoColor=white"> <img src="https://img.shields.io/badge/Python-66B2FF?style=for-the-badge&logo=Python&logoColor=white">
모든 명령어가 에흑으로 이루어진 난해한 프로그래밍 언어
<br><br>

## IDE 사용하기
[릴리즈 페이지](https://github.com/scott2010-gh/ehuk/releases/tag/v.1.0.0-beta)로 이동하여 IDE 다운로드가 가능합니다.

## 명령어 목록
명령어 | 기능
|---|---|
에>흑 | 포인터 증가
에<흑 | 포인터 감소
에에훅 | 포인터 위치의 값 1 증가
에흑 | 포인터 위치의 값 1 감소
에!흑 | 포인터 위치의 값 출력
에?흑 | 입력된 값을 포인터 위치에 저장
에_흑 | 띄어쓰기 출력
에@흑[ | 반복문 시작
에@흑] | 반복문 종료
에;흑{ | 주석 시작
에;흑} | 주석 종료

***각 명령어 사이에는 항상 공백이 입력되어야 합니다***


## python 구현체 사용법

#### 초기화
`compiler = 에흑(5)`<br>
메모리 크기를 5로 설정하여 초기화(기본값 32768)

#### 입출력 설정
`compiler.pm=1`<br>
아스키 코드 출력모드로 설정(기본값)
<br>
`compiler.pm=0`<br>
숫자 출력모드로 설정
<br>
`compiler.im=1`<br>
숫자 입력모드로 설정(기본값)
<br>
`compiler.im=0`<br>
아스키코드 입력모드로 설정

#### 프로그램 로드 및 실행
```python
compiler.load(path)
compiler.reset()
compiler.run()
```
> 예) 숫자 출력모드, 아스키코드 입력모드로 C:\test.에흑 실행
> - compiler.load("C:\\test.에흑")
> - compiler.pm=0
> - compiler.im=0
> - compiler.reset()
> - comoiler.run()

<br>

#### 메모리 확인(디버깅용)
`compiler.memory()`<br>
메모리 리스트 출력<br>

#### 반복 쓰기 도우미
`compiler.hw(text,num)`<br>
text를 num번 이어서 출력<br>
> 예) '에에흑 '을 15번 출력
> - compiler.hw('에에흑 ',15)

<br>

## 예시 코드

#### 1. 값을 다른 위치로 이동시키기

```에흑
에에흑 에에흑 에에흑 에에흑 에에흑 
에@흑[ 에흑 에>흑 에에흑 에<흑 에@흑] 
```
**설명**<br>
`에에흑 에에흑 에에흑 에에흑 에에흑`<br>
초기 포인터 위치(0)의 값을 5로 설정합니다.<br><br>
`에@흑[ 에흑 에>흑 에에흑 에<흑 에@흑]`<br>
0 위치의 값이 0이 아닌동안
>- 포인터 위치(0)의 값 1 감소
>- 포인터 위치 1 증가(위치 : 1)
>- 포인터 위치(1)의 값 1 증가
>- 포인터 위치 1 감소(위치 : 0)

를 반복합니다.

-> 결과적으로 0 위치의 값 5가 1 위치로 이동합니다.

<hr>

#### 2. 두 값 더하기
```에흑
에흑 에흑 에흑 에흑
에>흑
에흑 에흑 에흑
에@흑[ 에흑 에<흑 에에흑 에>흑 에@흑]
```
**설명**<br>
`에흑 에흑 에흑 에흑 `<br>
0 위치의 값을 4로 설정합니다<br><br>
`에>흑 `<br>
포인터 위치를 1 증가시킵니다(위치 : 1)<br><br>
`에흑 에흑 에흑 `<br>
1 위치의 값을 3으로 설정합니다<br><br>
`에@흑[ 에흑 에<흑 에에흑 에>흑 에@흑]`<br>
1 위치의 값이 0이 아닌동안
>- 포인터 위치(1)의 값 1 감소
>- 포인터 위치 1 감소(위치 : 0)
>- 포인터 위치(0)의 값 1 증가
>- 포인터 위치 1 증가(위치 : 1)

를 반복합니다.<br>
-> 결과적으로 0위치의 값 4와 1위치의 값 3이 더해져 0 위치에 저장됩니다.
